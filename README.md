Тема проекта: Разработка модели бинарной классификации для задачи кредитного скоринга.

Цель: Построить модель, прогнозирующую вероятность дефолта клиента по кредиту (задача бинарной классификации) с высоким качеством метрики ROC AUC.

Этапы работы:

Разведочный анализ данных (EDA):

Проведен автоматизированный анализ 90 числовых признаков с помощью библиотеки ydata_profiling. Это позволило быстро получить визуализацию распределений, матрицу корреляций и выявить проблемы в данных (выбросы, пропуски, дисбаланс).

Отбор признаков (Feature Selection):

Для уменьшения размерности и улучшения качества модели использован алгоритм DecisionTreeClassifier для оценки важности признаков (Feature Importance). В финальную выборку вошли признаки, вносящие не менее 1% информации.

Предобработка данных (Data Preprocessing & Feature Engineering):

Обработка пропусков: Числовые переменные заполнены медианными значениями.

Создание новых признаков:

Категориальный признак «Стаж работы» преобразован в интервалы (биннинг): "<1 года", "1-3 года", "3-5 лет" и т.д.

Из признака «Дата первого займа» извлечено десятилетие, что позволило косвенно оценить возрастную когорту клиента.

Моделирование:

В качестве финальной модели выбран градиентный бустинг CatBoostClassifier, так как он эффективно работает с разнородными признаками (категориальными и числовыми) и не требует прямого кодирования (One-Hot Encoding) для категорий.

Результаты:

Итоговая модель достигла значения метрики ROC AUC = 0.759 на тестовой выборке.

Данный результат оказался на 15% выше, чем у модели логистической регрессии (бейзлайн), и позволил занять 25 в соревновании.
