Проект построения модели бинарной классификации для кредитного скоринга.
В начале был проведен EDA при помощи библиотеки ydata_profiling для числовых переменных. Результатом работы библиотеки является html файл. В html файле имеется:
Анализ каждого показателя с визуализацией распределения
Матрица корреляций
Уведомления о возможных проблемах с данными
По результатам работы библиотека показала себя очень хорошо для EDA большого количества признаков (90) для задачи
Для построения я решил отобрать признаки при помощи Дерева решений, которое указывает % информации, который дает каждый признак. Были выбраны признаки, которые дают хотя бы 1%.
Далее была работа с данными, после которой:
Числовые переменные были заполнены (Либо медианой для числовым, определенным % для % переменных)
Категориальные перемены были сильно перекодированы:
Стаж был перекодирован в года (<1 года, 1 год ... )
Также дата первого займа была перекодирована по десятилетиям ( В целом параметр показывает примерный возраст клиента )
Для обучения была выбрана модель CatBoostClassifier так как в итоговой моделе осталось много категориальных признаков
Результатом соревнования была метрика ROCAUC. Модель выдала результат 0.759, что в целом неплохо.
